# 💧 Healthy Water Bottles Website

## 📖 Overview
This project is a simple website design for showcasing and selling eco-friendly, reusable water bottles.  
The focus is entirely on **frontend design**, created with **HTML5 and CSS3**, to deliver a clean, modern, and responsive user interface.  
Currently, the project covers only the **design phase** without shopping cart or backend functionality.

---

## 🚀 Features
- Modern, responsive design that works on mobile, tablet, and desktop.  
- Home page with product listings, images, and prices.  
- Simple navigation between pages (Home, Products).  
- Clean typography and color scheme to highlight the eco-friendly theme.  

---

## 🛠️ Technologies Used
- **HTML5** → Semantic structure and accessibility.  
- **CSS3** → Styling, responsive layout, and visual design.
